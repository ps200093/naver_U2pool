# 네이버 쇼핑 가격 비교 크롤러

Selenium을 활용한 네이버 쇼핑 가격 비교 크롤링 프로젝트입니다.

## 📋 프로젝트 소개

네이버 쇼핑에서 상품 검색 시 나타나는 가격 비교 정보를 자동으로 수집하는 크롤러입니다.

## 🚀 주요 기능

- 네이버 쇼핑 상품 검색
- 상품 정보 수집 (상품명, 가격, 판매처 등)
- 가격 비교 데이터 추출
- 결과 데이터 저장 (CSV, JSON 등)

## 🛠️ 기술 스택

- **Python 3.x**
- **Selenium** - 웹 자동화 및 크롤링
- **Chrome WebDriver** - 브라우저 제어

## 📦 설치 방법

### 1. 필수 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. Chrome WebDriver 설치

#### 자동 설치 (권장)
프로젝트에서 `webdriver-manager`를 사용하면 자동으로 WebDriver가 설치됩니다.

```bash
pip install webdriver-manager
```

#### 수동 설치
또는 [Chrome WebDriver 다운로드 페이지](https://chromedriver.chromium.org/downloads)에서 Chrome 버전에 맞는 WebDriver를 다운로드하여 설치합니다.

## 💻 사용 방법

### 기본 실행

```python
python main.py
```

### 특정 상품 검색

```python
python main.py --keyword "검색할 상품명"
```

## 📂 프로젝트 구조

```
naver_U2pool/
├── main.py              # 메인 실행 파일
├── crawler.py           # 크롤링 로직
├── parser.py            # 데이터 파싱
├── utils.py             # 유틸리티 함수
├── requirements.txt     # 필요한 패키지 목록
├── config.py            # 설정 파일
├── data/                # 크롤링 결과 저장 폴더
└── README.md            # 프로젝트 설명서
```

## ⚙️ 설정

`config.py` 파일에서 다음 설정을 변경할 수 있습니다:

- 크롤링 대기 시간
- 저장 파일 형식
- 브라우저 headless 모드 설정
- 최대 수집 상품 개수

## 📊 출력 데이터 형식

수집되는 데이터 항목:

- 상품명
- 가격
- 판매처
- 배송비
- 리뷰 수
- 상품 링크
- 이미지 URL

## ⚠️ 주의사항

1. **법적 책임**: 크롤링 시 네이버의 이용약관 및 robots.txt를 준수해야 합니다.
2. **요청 간격**: 서버에 부담을 주지 않도록 적절한 대기 시간을 설정하세요.
3. **개인정보**: 수집한 데이터의 개인정보 보호법 준수가 필요합니다.
4. **상업적 이용**: 상업적 목적으로 사용 시 네이버의 사전 허가가 필요할 수 있습니다.

## 🔧 문제 해결

### WebDriver 오류
- Chrome 브라우저와 WebDriver 버전이 일치하는지 확인하세요.
- `webdriver-manager`를 사용하여 자동으로 버전을 관리하는 것을 권장합니다.

### 크롤링 실패
- 네이버 페이지 구조가 변경되었을 수 있습니다.
- 네트워크 연결 상태를 확인하세요.
- 대기 시간을 늘려보세요.
